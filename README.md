# Projekts: Skolēnu datu vizualizācija

Šī ir tīmekļa lietotne, kas izveidota ar Python un Flask. Lietotne ļauj augšupielādēt CSV failu, analizēt un vizualizēt datus, kā arī saglabāt tos datubāzē.

## 💻 Lietotās tehnoloģijas

- Python
- Flask
- Pandas
- matplotlib
- SQLite

## 🚀 Funkcijas

- CSV datu augšupielāde un nolasīšana
- Datu vizualizācija (diagrammas un histogrammas)
- Datu saglabāšana SQLite datubāzē
- Lietotāja saskarne ar HTML (Flask Templates)

## 📂 Projekta faili

- `app.py` — galvenā Flask lietotne
- `data.csv` — piemēra dati
- `db.py` — datubāzes pieslēgums
- `README.md` — apraksts
